Etiquetas HTML vistas en clase

Nombre Descripción Ejemplo
html Define el documento de HTML <html> DOM</html>
body Define el cuerpo de documento <body> DOM</body>
head Define el elemento del encabezado correspondiente al documento html <head> DOM</head>
strong Define el inicio y final de un párrao de texto. <p>aquí el prárrafo</p>
p Fragmento de texto importante o palabras clave. <strong> texto importante</strong>
blockquote Agrupa información y características de una cita. <blockquote> cita importante</blockquote>
i Fragmento de texto con voz o tono alternativo al resto. <strong> texto importante</strong>
b Fragmento de texto destacable en negrita. <b> texto importante</b>
u Fragmento de texto para nombres propios o escritura incorrecta intencionada <b> texto importante</b>
a Etiqueta para hacer referencia a otras páginas web o recursos como documentos o archivos. <s> recurso importante</s>
href Atributo obligatorio Enlace al documento que se quiere cargar. <br />
br Salto de línea, es una etiqueta con autoclausura. <br />
hr Indica una separación por temas del texto <br />

Etiquetas de Tablas

Nombre Descripción Ejemplo
table Etiqueta contenedora que tendrá en su interior toda la tabla. <table>encabezado</table>
thead Table Head Etiqueta contenedora de la cabecera de la tabla. Parte superior de la tabla. <thead>titulo</thead>
tbody Table Body Etiqueta contenedora del cuerpo de la tabla. Parte central de la tabla. <thead>titulo</thead>
th Table Header. Cada una de las celdas de cabecera de la tabla. <table>encabezado</table>
tr Table Row. Etiqueta contenedora de cada fila de la tabla. <table>encabezado</table>
td Table Data. Cada una de las celdas de la tabla. <table>encabezado</table>
colspan Número de columnas que la celda abarcará. <table>encabezado</table>
rowspan Número de filas que la celda abarcará. <table>encabezado</table>

Etiquetas de Listas

Nombre Descripción Ejemplo
ul Define una lista sin orden. Se trata de la etiqueta contenedora. <ul>elementos</ul>
ol Define una lista numerada (con orden). Etiqueta contenedora. <ol>elementos</ol>
li Define un ítem de la lista. <li>elementos</li>

Etiquetas para imágenes

Nombre Descripción Ejemplo
img Define la ubicación de una imagen y su descripción. <img src="./img/river-1.jpg" alt="Rio en Canadá">
src Atributo obligatorio Indica el nombre o la URL de la imagen a mostrar. src="./img/river-1.jpg"
alt Atributo obligatorio Establece un texto alternativo que describa la imagen a mostrar. alt="Rio en Canadá"
