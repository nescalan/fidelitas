Es un sitio web donde se almacena información sobre la compatibilidad de los navegadores(de escritorio, móviles, etc) con diversas tecnologías web HTML5, CSS3, SVG.

css3test

Es un excelente tester en linea que te ayuda a determinar la compatibilidad de tu navegador con el borrador CSS3, proporcionado el valor del tanto porcentual de dicha compatibilidad y una lista detallada de las pruebas que se hicieron.
